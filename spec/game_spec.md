# 一画面パズルRPG『ロードガイド勇者 〜 パネル配置道繋ぎRPG 〜』ゲーム企画・仕様定義書

## 1. ゲーム概要

### 1.1 コンセプト
* **タイトル**: ロードガイド勇者 〜 一画面パネル道繋ぎRPG 〜
* **ジャンル**: カジュアル・一画面ルート構築パズルRPG
* **ターゲット**: 手軽に楽しみたいけれど、適度な思考と工夫による爽快感が好きなカジュアルゲーマー
* **ゲームの根幹**: プレイヤーは、行く手を阻むトラップ（毒の沼地）を上手く避けながら、お城から魔物を討伐するための最適なルートを、有限配布される「道パネル」をやりくり（DIY）して配置するパズルRPGです。複雑すぎないカジュアルな遊びやすさと、パズルとしての戦略性と爽快さを両立します。

---

## 2. 主要画面・システム構成

本ゲームはユーザー体験の快適さと一覧性を極限まで高めるため、スクロール不要で全体を瞬時に把握できる**「完全一画面（Non-Scrollable Layout）」**構成を採用しています。

### 2.1 画面ビュー構成
1. **ステージセレクトビュー（Stage Select View）**
   * 解放済みのステージカードが一覧表示され、プレイヤーが挑戦するレベルを選択します。
   * クリア状況（星マーク）がローカルストレージ（`localStorage`）を介して保存・同期されます。
2. **メインゲームプレイビュー（Main Gameplay View）**
   * ステージに突入した後の、パズルの構築および勇者の進行を司る実開発用メイン画面です。

## 3. コアループ & ゲームルール

```
[ 各パネルの残り枚数（在庫）を確認 ] 
        ↓
[ 在庫から道パネルを選択して、盤面の空きマスに配置 ] 
        ↓
[ 毒の沼地（デバフ）を回避・鍛冶屋や宝箱を経由するルートを模索 ]
        ↓
[ 配置したパネルを回転・撤去してルート最適化 ]
        ↓
[ 進軍（GUIDE）開始：戦闘力を道中で爆発的に強化 ]
        ↓
[ 全ての指名手配モンスターを撃破 ] ──（敗北/道塞がり）──→ [ 再配置・レベル再考 ]
        ↓
[ ステージクリア ＆ 最終ステータスが次のステージへと蓄積・引き継ぎ！ ]
```

### 3.1 枚数リソース制（ストック有限）パネルシステム
手札のランダムドロー制を廃止し、ステージごとにあらかじめ決められた道パネルの在庫数をやりくりして進路を構築する**「枚数リソース（ストック有限）制」**を採用します。

* **在庫数の設定例**:
  * ステージ開始時に「直進：残り4枚」「曲がり（L字）：残り3枚」「T字：残り1枚」といった上限数が設定されます。
  * プレイヤーはUI上のパネル在庫一覧から、配置したいパネルを自由に選んで盤面に配置します。
  * 配置すると、該当パネルの在庫数が「-1」されます。
  * 盤面のパネルを撤去（お片付け）すると、在庫が「+1」されて手元に戻ります。
  * これにより、不要な配牌に悩まされる運要素をなくし、「限られた資材をどうやりくりして最適なルートを作るか」という純粋なパズル＆パズル構築のみに集中して楽しめます。

### 3.2 盤面配置・操作システム（Game Grid Board）
* ステージに応じてグリッド幅が適用されます（3x3、4x4、5x5、6x6など）。
* 盤面には既に「モンスター」や「鍛冶屋」などの**固定固有マス（静的プリセット）**が配置されており、これらを迂回または必ず経由するように道を繋ぎます。
* **空きマスをクリック**: 選択しているパネルがスロットに嵌め込まれます（在庫から1枚消費）。
* **配置済みパネルをクリック**: 対象のタイルが**90度右回転**します。
* **撤去（ごみ箱）**: 配置されたパネルの上にホバーすると表示される「ゴミ箱ボタン」を押すと、パネルを撤去して在庫を回収し、再び空きマス（闇）に戻せます。
* **現在位置＆履歴のロック機能**: 勇者が進軍（歩行）を開始すると、勇者の現在地および既に通過した「踏破済みパネル」は、回転や撤去が一切不可能な**「ロック状態🔒」**になります。

### 3.3 施設およびお邪魔ギミック（トラップ）
道中で勇者が通過すると即時、またはセンター到達時に発動する特殊効果パネルです。

| 施設・ギミック名 | アイコン/呼称 | 主な効果 | 計算ロジック・調整方針 |
| :--- | :--- | :--- | :--- |
| **王城（Castle）** | `Castle` | スタート地点。ステージごとの指定位置に配置。 | 前ステージから引き継いだ戦闘力（POW）ベース値で出撃します。 |
| **鍛冶屋（Forge）** | `Sword` / `[+鍛冶屋]` | 勇者に剣・斧などを提供。 | 戦闘力（POW）を**固定値プラス**する（例: `+8`、`+25` など） |
| **修練所（Dojo）** | `Flame` / `[x道場]` | 勇者が肉体を極限まで鍛え上げる。 | 戦闘力（POW）を**割合乗算**する（例: `*1.5`、`*1.8` など） |
| **ショップ（Shop）**| `ShoppingBag` / `[+薬店]` | 秘薬を飲む。 | 戦闘力（POW）を**固定値プラス**する（例: `+15`、`+40` など） |
| **聖泉（Fountain）**| `Sparkles` / `[*聖泉]` | 神聖な泉の水を飲む。 | 戦闘力（POW）を**大幅割合乗算**する（例: `*2.0` など） |
| **毒の沼地（Poison）** | 紫色の濁った不快な沼 | 通過すると能力が低下するお邪魔マス。 | **難易度に応じた二段式のデバフ設計**:<br>① **通常ステージ**: 戦闘力を**固定値で引き算**する（例: `POW -50` などの大ダメージ）。<br>② **後半の難関ステージ**: 戦闘力を**割合で引き算（デバフ低下）**する（例: `POW × 0.8` 減衰）。<br>お邪魔でありつつも、盤面拡大に伴い「わざと通り抜けてショートカットするか」「完全迂回するか」という絶妙な思考を促します。 |
| **指名手配（Enemy）**| 各種固有モンスター | ステージ上に潜む討伐対象。 | 遭遇時の勇者のPOW ≧ 魔物力の場合、魔物を**討伐成功**として除去。一歩でも下回れば**敗北（Defeated）**。 |

### 3.4 恒久的な成長（シンプル・ステータス引き継ぎ）
* 前のレベルをクリアした瞬間の「勇者の最終戦闘力（POW）およびステータス」が、次のステージでの初期戦闘力（王城からスタートする際のベース値）に**そのままダイレクトに蓄積され、引き継がれます。**
* **カジュアルならではのインフレの爽快感**: 
  * 前のステージでどれだけ神がかったルートを構築し、勇者を限界突破させてクリアしたかが、そのまま次のステージの初期強さに直結します。
  * プレイヤー自身のパズルスキル（ルート構成の腕前）が勇者の「永続強化」へと変換されるため、シンプルながらも非常に高い達成感と、同一勇者への育成・愛着を感じることができます。

## 4. 進軍（歩行）・AI追跡規則

1. **出撃命令（GUIDE）**: プレイヤーが画面上の「GUIDE（進軍開始）」ボタンを押すと歩行開始。いつでも一時停止や進軍速度の変更（1x / 1.5x / 2.0x）が可能です。
2. **接続判定アルゴリズム**:
   * 勇者がタイルの中央に到達すると、現在進行方向の「出口ポート」と、隣接するタイルの「入口ポート」が一致（接続状態）しているかをチェックします。
   * **道が繋がっている場合**: 接続ポートを通じて次のタイルへ自動でスムーズに移行。
   * **道がない・矛盾している場合**: 勇者は立ち止まり、ステータスが「Blocked（進行不能）」になります。
3. **歴史トラッキング**: 
   * 勇者が移動したタイルのグリッド座標は `pathHistory` に格納されます。一度訪れた効果マスは一度踏んだ時点で効果を使用済みに切り替わります。

## 5. ステージ構成・難易度設計イメージ

* **Stage 1 (旅立ちの草原 - 4x4)**: 基本ルール（接続と攻撃力強化）習得。鍛冶屋・ショップを巡り、ポーションと剣でスライムを討伐。
* **Stage 2 (毒沼の古代遺跡 - 4x4)**: お邪魔ギミック「毒の沼地（固定値引き算デバフ）」が初登場。避けるべきか、武器のプラス値で相殺して強行突破するかを選択。
* **Stage 3 (修練の溶岩谷 - 5x5)**: グリッドがやや広がり、道場（乗算）を活用する必要が出現。引き継いだ強力なステータスを用いて大敵を薙ぎ倒す。
* **Stage 4 (聖泉と魔風の空中回廊 - 5x5)**: 毒の沼地のデバフが「割合引き算（POW × 0.8）」へと凶悪化。聖泉（2.0倍）と組み合わさり、通過順の掛け合わせが重要な高度ルート設計へ。
* **Stage 5 (魔王ルシファーの御座 - 6x6)**: 最大盤面。これまで何ステージ分も重ねて引き継ぎ限界突破してきた最強の勇者で、あらゆる武器・道場・沼地を計算尽くした長大な一本道を築き、魔王を撃破する大団円。
